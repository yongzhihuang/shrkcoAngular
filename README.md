## Quick Start

Install Node.js and then:

```sh
$ git clone git://github.com/joshdmiller/ng-boilerplate
$ cd ng-boilerplate
$ sudo npm -g install grunt-cli karma bower
$ npm install
$ bower install
$ grunt watch
```


## Development Workflow

Starts a server while watching files
```
grunt watch
grunt server
```

Build for production
```
grunt
```
